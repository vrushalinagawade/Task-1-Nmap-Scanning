# Task 1: Basic Network Scanning with Nmap

## Objective
To identify open ports and services on a local machine using Nmap.

## Tool Used
- Nmap 7.98
- Zenmap GUI

## Command Executed
nmap -sV 127.0.0.1

## Observations
Several system services were found running on the local machine, indicating active network services.

## Conclusion
Nmap helps identify exposed services and is a critical tool in vulnerability assessment and penetration testing.
